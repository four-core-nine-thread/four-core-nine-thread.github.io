JDBC简介和作用

JDBC(Java Database Connectivity),中文名为java数据库连接，是SUN公司提供的使用Java访问数据库的应用程序接口，让程序员可以通过Java来进行数据库操作，现在的JDBC可以在Oracle官网进行下载。Python中像mysqldb,pymysql也是实现了PEP提供的数据库访问规范，只是在使用时简化了过程，没有像JDBC那种通过反射来注册驱动。
