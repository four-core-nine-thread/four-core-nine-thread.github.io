发表于 2018-02-10 | 分类于 Java ， 数据库

jdbcTemplate简介

jdbcTemplate是Spring框架针对jdbc的封装,他是一个基于ORM思想的jdbc工具类，简化了直接使用jdbc，数据库连接池的代码，提供了更加方便的操作来进行数据库操作。jdbcTemplate在Spring-jdbc的jar包中，并且还需要使用到Spring-tx的jar包（和事务相关），当然jdbc,连接池的jar包也必不可少。
